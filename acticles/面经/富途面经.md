Q：给一个数组，把数组的元素打乱一下，每个元素出现的位置完全随机（每个位置出现的概率完全相等）。要求空间复杂度为 O(1)，时间复杂度为  O(n).

比如：[1, 2, 3, 4, 5] ==> [2, 4, 1, 5, 3]，每个位置的概率一样。

A：

``` go
func randSort(arr []int) {
    n := len(arr)
    for i:=0; i<n; i++ {
        rd := i+rand.Intn(n-i)
        arr[i], arr[rd] = arr[rd], arr[i]
    }
}
```
